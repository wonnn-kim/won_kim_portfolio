Project Procedural Geometric Modeling

3D Graphics

Project Overview

This project focuses on procedural mesh generation of basic 3D primitives such as sphere, cylinder, cone, torus, and plane.

Tasks Completed

  • Sphere - Generated a UV-mapped sphere be deviding stacks and slices.
  • Cylinder - Surface was constructed by forming triangulated mes with consistent topology. The top and boottom caps were created using fan-based indexing.
  • Cone - Similar to Cylinder, but generated sloped sides with vertex normals.
  • Torus - Generated tours using parametric equations and normals.
  • Plane - Implemented plane which vertex's position and UV coordinate were linearly interpolated.
  • Designed vertex layout structures including position, normal, and UV.
  • Ensured every primitive used consistent topological rules and tessellation logic.

Reflection

Ensuring consistent vertex layout and UV alignment across different shapes was quite challenging.

Writing each shape from scratch taught me how real-time graphics engines handle procedural assets.
